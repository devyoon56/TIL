# 서브스크립트의 확장

타입을 확장하여 서브스크립트를 추가할 수 있습니다.

## 서브스크립트의 확장 예시

Int 타입을 확장하여 서브스크립트를 추가하는 예제입니다. 아래 서브스크립트의 경우 인스턴스에서 서브스크립트를 호출하면 10의 n 제곱의 자리수를 반환하는 서브스크립트입니다.

```swift
extension Int {
   subscript(num: Self) -> Self {
       var decimalBase = 1

       for _ in 0..<num {
           decimalBase *= 10
       }

       return (self / decimalBase) % 10
   }
}

123456789[0]        // 10의 0 제곱의 자리수: 9
123456789[1]        // 10의 1 제곱의 자리수: 8
123456789[2]        // 10의 2 제곱의 자리수: 7
```
