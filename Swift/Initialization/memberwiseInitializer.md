# 멤버와이즈 생성자(Memberwise Initializer)

구조체도 클래스처럼 기본 생성자 init()이 존재합니다. 하지만 구조체에는 클래스에 없는, 구조체만의 특별한 생성자가 존재합니다. 이 특별한 생성자는 **멤버와이즈 생성자**라고 합니다. 구조체의 **멤버와이즈 생성자**는 기본 생성자처럼 자동으로 생성되는 생성자입니다.

구조체에서 모든 저장 속성들을 선언할 때 초기값을 할당하더라도 특이하게 멤버와이즈 생성자를 자동으로 제공합니다. 하지만 직접 별도의 생성자를 구현하면 멤버와이즈 생성자는 자동으로 제공되지 않습니다. 멤버와이즈 생성자는 개발 편의를 위한 편의적 기능이라고 생각하면 됩니다.

## 멤버와이즈 생성자 예시

```swift
struct Color {
    var red: Double = 255.0
    var green: Double
    var blue: Double
}

let white = Color(green: 255.0, blue: 255.0)        // 멤버와이즈 생성자 호출
let black = Color(red: 0.0, green: 0.0, blue: 0.0)  // 멤버와이즈 생성자 호출
```

이렇게 자동으로 생성되는 멤버와이즈 생성자를 호출해서 인스턴스를 생성하면 됩니다.
