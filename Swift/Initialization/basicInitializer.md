# 초기화(Initialization)와 생성자(Initializer)

초기화는 커스텀 타입인 클래스, 구조체 또는 열거형의 인스턴스를 생성하는 것을 말합니다. 초기화는 모든 **저장 속성**에 대한 초기값을 설정하여 인스턴스를 생성하고 이렇게 생성한 인스턴스를 사용 가능한 상태로 만드는 것으로 요약할 수 있습니다.

> 열거형의 경우 저장 속성이 존재하지 않기 때문에 열거형 케이스를 사용하여 인스턴스를 생성합니다.

그렇다면 생성자는 무엇일까요? 생성자는 인스턴스를 생성할 때 인스턴스의 모든 저장 속성에 초기값을 갖게하는 특별한 메서드를 말합니다. 즉 생성자의 역할은 인스턴스 생성 시 인스턴스의 모든 저장 속성이 초기값을 갖도록 하는 것입니다.

> 생성자의 실행이 끝나는 시점에 모든 저장 속성은 초기값을 반드시 가져야 합니다.

생성자는 그 역할 때문에 초기화 메서드라고도 합니다. 그리고 특별한 메서드이기 때문에 **init** 이라는 메서드 이름을 가집니다. **init**은 initializer의 줄임말이라고도 볼 수 있습니다.

## 생성자 기본

```swift
class Color {
    let red: Double
    let green: Double
    let blue: Double

    // 기본 생성자
    init() {
        red = 0.0
        green = 0.0
        blue = 0.0
    }

    init(white: Double) {
        red = white
        green = white
        blue = white
    }

    init(red: Double, green: Double, blue: Double) {
        self.red = red
        self.green = grenn
        self.blue = blue
    }
}

let color1 = Color()        // 기본 생성자를 호출하여 초기화
color1.red                  // 0.0
color1.green                // 0.0
color1.blue                 // 0.0

let color2 = Color.init()   // 이렇게도 기본 생성자를 호출할 수 있습니다.

let colorWhite = Color(white: 255.0)      // 생성자를 호출하여 초기화
// let colorWhite = Color.init(white: 255.0) 도 가능합니다.

let colorBlack = Color(red: 0.0, green: 0.0, blue: 0.0)       // 생성자를 호출하여 초기화
// let colorBlack = Color.init(red: 0.0, green: 0.0, blue: 0.0) 도 가능합니다.
```

- init()은 기본 생성자라고 합니다.

  - 간단하게 위 코드의 기본 생성자와 같이 모든 저장 속성에 원하는 값을 초기화하여 인스턴스를 생성할 수 있습니다.
  - 모든 저장 속성을 선언할 때 값을 할당하면 컴파일러는 기본 생성자를 자동으로 생성합니다.
  - 또는 저장 속성을 옵셔널로 선언하면 초기값이 없더라도 nil로 초기화되므로 기본 생성자가 자동으로 생성됩니다.
  - 따라서 저장 속성 선언 시 값을 모두 할당하거나 저장 속성을 옵셔널로 선언하면 기본 생성자를 구현할 필요가 없습니다.

> 별도로 생성자를 구현하지 않아도 기본 생성자를 자동으로 생성할 수 있는 조건이면 기본 생성자가 자동으로 생성되기 때문에 반드시 생성자를 정의할 필요는 없습니다. 하지만 위 코드처럼 기본 생성자 외에 다른 생성자를 구현한다면 기본 생성자는 자동으로 생성되지 않으므로 기본 생성자를 구현하고 싶다면 별도로 구현해야 합니다.

- init(white:), init(red:green:blue:)처럼 생성자도 오버로딩을 지원합니다.

  - 파라미터 수, 아규먼트 레이블, 자료 타입으로 생성자를 구분합니다.

- 위 코드처럼 원하는 생성자를 호출해서 저장 속성을 초기화하고 클래스 인스턴스(객체)를 생성하면 됩니다.
