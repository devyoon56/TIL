# 속성(Property)

구조체와 클래스 정의 내부에서 선언한 변수와 상수를 속성 또는 프로퍼티라고 합니다. 구조체와 클래스의 속성은 동일합니다. 속성이 동일하다는 의미는 속성의 종류가 같고 속성의 특징이 구조체와 클래스에 동일하게 적용된다는 것을 의미합니다.

## 속성의 종류

- 저장 속성(Stored Property)
- 지연 저장 속성(Lazy Stored Property)
- 계산 속성(Computed Property)
- 타입 속성(Type Property)
- 속성 감시자(Property Observers)

속성 감시자는 속성의 종류는 아니지만 일반적으로 저장 속성의 값이 바뀌는 것을 감시하는 역할을 하는 메서드로, 속성에 대해 공부하면서 정리하겠습니다.
