# while 반복문

while 반복문은 조건이 false가 될 때까지 while 반복문 내부를 실행합니다. while 반복문은 반복문 시작 전에 반복 횟수를 알 수 없을 때 일반적으로 사용합니다.

## while 반복문의 형식

while 반복문은 먼저 조건을 평가합니다. 조건이 **true**이면 반복문을 실행하고 조건이 **false**가 될 때까지 반복문을 반복합니다.

```swift
while 확인할 조건 {
    // code
}
```

## while 반복문 사용 시 주의할 점

while 반복문의 조건이 **false**가 되지 않는다면 반복문은 계속 실행하여 _무한루프_ 상태에 빠집니다. _무한루프_ 상태를 빠져나오기 위해 반복문의 조건을 **false**로 바꾸는 코드가 필요합니다.

```swift
var sum = 0
var num = 1

while num <= 50 {   // 조건이 만족하는 동안 계속 반복
    sum += num
    num += 1      // 이런 조건이 필요 (위의 조건식을 변화시키지 않으면 무한 반복하게됨)
}

print("총 합의 출력: ", sum)
```
