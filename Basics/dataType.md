# 데이터 타입

## 데이터 타입은 왜 필요할까?

- 사람은 데이터를 보면 직관적으로 어떤 타입인지 바로 알 수 있습니다.
- 하지만 컴퓨터는 데이터가 어떤 타입인지 직관적으로 알 수 없습니다.
- 따라서 컴퓨터에게 이 데이터가 어떤 형태인지, 얼마만큼의 크기로 저장할 것인지를 알려주어야 합니다.
- 이에 대해 약속한 것이 데이터 타입입니다.

## 스위프트의 대표적인 데이터 타입 종류

- 정수형 타입: Int
- 실수형 타입: Float, Double
- 문자형 타입: Character, String
- 불리언 타입: Bool

스위프트의 모든 데이터 타입은 항상 대문자로 시작합니다.

## 정수형 타입

- 정수는 '분수 성분이 없는 수'를 말합니다.
- 부호가 있는 것(signed)과 부호가 없는 것(unsigned)로 나뉩니다.
- 대부분의 경우에서 **Int** 데이터 타입을 사용하기 때문에 코드에서 사용할 특정 정수의 크기를 고를 필요가 없습니다.

## Int

- 특정 크기의 정수와 작업하는 것이 아니라면 코드에서 정수 값으로 항상 **_Int_**를 사용합니다.
  - 코드 일관성과 상호 호환성을 높일 수 있습니다.
- **Int**는 '현 플랫폼 고유의 워드 크기'와 같은 크기를 가집니다.
  - 32비트 플랫폼: Int32와 같음(32비트 정수형 타입)
  - 64비트 플랫폼: Int64와 같음(64비트 정수형 타입)
  - 아이폰 5s부터 64비트 CPU 사용
- **Int**로도 충분한 범위의 정수를 저장할 수 있습니다.

스위프트는 기본적으로 정수형 타입으로 **Int**를 사용합니다.

```swift
var peopleInTeam = 3
type(of: peopleInTeam)  // Int
```

## 실수형 타입

- 부동-소수점 수는 '분수 성분을 가진 수'를 말합니다.(3.14159, 0.01 ...)
- 부동-소수점 타입은 특성상 정수 타입보다 훨씬 넓은 범위의 값을 표현할 수 있습니다.
- 스위프트는 두 개의 부호 있는(signed) 부동-소수점 타입을 제공합니다.
  - Double: 64비트 부동-소수점 수
  - Float: 32비트 부동-소수점 수

## Float과 Double

Float

- 4바이트(32비트) 크기의 실수형 타입입니다.
- 전체 6자리의 소수점을 표현할 수 있습니다.(6자리의 유효 숫자)

Double

- 8바이트(64비트) 크기의 실수형 타입입니다.
- 전체 15자리의 소수점을 표현할 수 있습니다.(15자리의 유효 숫자)

Float과 Double은 정밀도의 차이가 있습니다.

```swift
var latitude: Float = 16.666666     // 16.66667
var longitude: Double = 16.666666666666     // 16.666666666666
```

스위프트는 기본적으로 실수형 타입으로 **Double**을 사용합니다.

```swift
var todayTemperature = 24.3
type(of: todayTemperature) // Double
```

## 문자형 타입

- 문자와 문자열은 반드시 **쌍따옴표("")**안에 써야합니다.
- 스위프트는 두 개의 문자형 데이터 타입을 제공합니다.
  - Character
  - String

스위프트는 기본적으로 문자형 타입으로 **String**을 사용합니다.

```swift
var firstName = "태윤"
var lastName: Character = "김"

type(of: firstName)     // String
firstName = "윤태"      // 에러 발생
```

타입 안정성 때문에 Character 데이터 타입에 String 데이터 타입을 할당할 수 없습니다.

## 불리언 타입

- 참과 거짓을 다루는 데이터 타입입니다.
- 스위프트는 두 개의 불리언 타입을 제공합니다.
  - false
  - true
- 참과 거짓을 나타내는 불리언 타입은 프로그래밍의 제어 흐름에서 중요한 역할을 합니다.
  - 조건문
  - 반복문

```swift
let isSwiftGood = true

if isSwiftGood {
    print("I love Swift")
}

while isSwiftGood {
    print("I looove Swift")
}
```
